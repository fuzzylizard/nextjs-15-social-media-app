# OpenShare

This is a project to learn Next.js, React, and the associated libraries
like React Query, Prisma, Shadcn, Tailwind css, etc.

It is a social media app like X or Threads written in Next.js 15 and React.

## Requirements

* Node v20+

## Commands

### Run the app in dev mode

```npm run dev```

### To push DB schema changes

```npx prisma db push```

### To view the prisma studio

```npx prisma studio```
